# SteamDepotValidator

Retrieve depot information from [SteamDB](https://steamdb.info) and verify local files.

> [!IMPORTANT]
>
> It is recommend to use [DepotDownloader](https://github.com/SteamRE/DepotDownloader) to validate files if you have ownership to the depot on your account.
>
> This program should only be used when you lack access to the depot but still need to validate files.
>
> [SteamDB does not allow scraping/crawling.](https://steamdb.info/faq/#can-i-use-auto-refreshing-plugins-or-automatically-scrape-crawl-steamdb) Use this tool at your own risk.
>
